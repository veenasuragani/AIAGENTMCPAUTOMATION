# Playwright - AIAgentMCPAutomation

Setup and run tests:

```powershell
cd "C:\Users\Veenasri\Desktop\Automation\AIAgentMCPAutomation"
npm install
npx playwright install
npm test
```

- `playwright.config.js` - Playwright configuration
- `tests/example.spec.js` - example test
